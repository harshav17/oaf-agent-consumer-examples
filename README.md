## Setup oaf-agent-consumer-examples
```
npm install
```

## To run hex_adds example
```
npx ts-node src/hex_adds/index.ts
```

## To run an example in debug mode
```
DEBUG=oaf:* npx ts-node src/hex_adds/index.ts
```
